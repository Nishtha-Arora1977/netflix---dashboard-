# Netflix Interactive Dashboard 📊🎬

An interactive Netflix Dashboard built using Tableau to analyze Netflix movies and TV shows data.

## Features
- Movies vs TV Shows distribution
- Ratings analysis
- Top genres visualization
- Country-wise content analysis
- Year-wise growth of Netflix content

## Tools Used
- Tableau
- Excel / CSV
- Data Visualization
- Exploratory Data Analysis

## Dashboard Preview

![Dashboard Screenshot](netflix dashboard.png)

## Insights
- Movies dominate Netflix content library.
- TV-MA is the most common content rating.
- Documentaries and Stand-Up Comedy are among top genres.
- Significant growth in Netflix content after 2015.
